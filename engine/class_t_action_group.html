Detailed Description

A group of TAction that operate simultaneously. The class is a TAction itself.

The class provides methods for managing group of TAction, allowing to specify when finish: on success or failure of an action or never, waiting that all the actions have finished. When the TActionGroup is stopped or finish, all its (active) TAction are stopped too. Each TAction object is automatically deleted when it finishes.

The actions can be added with the addAction() method. The start() method will in turn execute all the added actions.
