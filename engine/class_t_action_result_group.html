Detailed Description

A group of TActionResult related to actions that operate simultaneously. The class is a TActionResult itself.

The class provides methods for managing group of TActionResult, allowing to specify when stop: on success or failure of an action or never, waiting that all the actions have finished. Each TActionResult object is automatically deleted when it has finished.
