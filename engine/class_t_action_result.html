Detailed Description

The result of an asynchronous operation.

The class is used to discriminate over a number of operations that use the same methods but involves different parameters. Each TActionResult has a finished signal that can be connected to a different slot. This allow, for example, to connect the parsing end of two data element to two diffent slots.

The idea is the same of the QNetworkAccessManager that return a QNetworkReply (the TActionResult) for each new network request.
